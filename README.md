# Educational attendance tracker
Проект по распознаванию количества посещений студентами аудиторий с использованием распознавания изображения с камер видеонаблюдения.
### Функционал системы
* распознавание количества студентов с видеофрагмента/потокового видео с ip камеры (количество + confidence)
* хранение и обработка данных (backend + БД)
* веб-дешборд для вывода статистики (по преподавателям/по группе/по дисциплине + загрузка расписания excel) (frontend)
### Стек

Проект выполнен в рамках производственной практики
